# 🎬 YouTube Homepage Clone

### 👋 Welcome to my repository! 

Thank you for stopping by! This project is a front-end pixel-perfect clone of the YouTube desktop homepage layout. 

I built this project completely **from scratch** to test, challenge, and showcase my core front-end development skills. It serves as a visual and structural demonstration of my mastery over semantic layouts, advanced CSS architectures, and responsive styling principles.

---

## 💡 Concepts Practiced & Demonstrated

Building a complex interface like YouTube requires a strong grasp of foundational and advanced layout techniques. Through this project, I successfully implemented:

* **Fluid Responsive Design:** Written custom CSS media queries (`@media`) to handle dynamic viewport scaling. The video layout seamlessly self-adjusts its grid column counts and interface elements depending on the browser tab or screen window size.
* **Semantic HTML5 Structuring:** Writing clean, readable, and structured markup. I wrapped video elements (like thumbnails, titles, and profile pictures) in interactive anchoring structures (`<a>` tags) mimicking native browser behavior.
* **CSS Grid Layouts:** Designed a flexible, multi-column video card grid framework (`vg` class) that scales smoothly across different device sizes.
* **CSS Flexbox Component Alignment:** Utilized Flexbox heavily to build the complex utility sections—including the left, middle, and right alignments inside the layout header (`.ls`, `.ms`, `.rs`) and sidebar layouts.
* **Component-Based CSS File Structure:** Organized styles across modular stylesheets (`general.css`, `video.css`, `header.css`, `sidebar.css`) to ensure code maintenance aligns with modern production environments.
* **Advanced Positioning & Tooltips:** Used absolute and relative positioning to construct custom tooltip UI overlays on interaction buttons (like Search, Create, and Notifications).

---

## 🛠️ Tech Stack Used

* **HTML5:** Semantic architecture, document structuring, and interactive layout handling.
* **CSS3:** Media Queries (Responsive Breakpoints), Grid/Flexbox mechanics, hover states, fixed positioning, typography formatting, and custom UI elements.
* **Google Fonts:** Integrated the official `Roboto` typography stack for an accurate platform look.
